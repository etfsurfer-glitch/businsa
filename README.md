# 엘리프 성성호수공원 — 모바일 분양 홍보 페이지

천안 성성지구 호수공원 프리미엄 단지 **엘리프 성성호수공원** 분양 홍보용 모바일 원페이지입니다.

## 구성
- `index.html` — 원페이지 마크업
- `styles.css` — 모바일 우선 스타일
- `app.js` — 히어로 슬라이더 / 탭 / 스크롤 인터랙션
- `assets/img/` — 본 단지 공식 자료 기반 이미지
- `_headers` — Cloudflare Pages 응답 헤더

## 배포
GitHub 연동 Cloudflare Pages가 `main` 브랜치 푸시 시 자동 배포합니다.
